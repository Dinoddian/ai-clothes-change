# AI服装换装应用

这是一个基于AI的服装换装应用原型，使用HTML、Tailwind CSS和JavaScript开发。

## 功能特点

- 照片上传/拍摄
- 服装选择
- AI换装生成
- 结果保存与分享

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome
- JavaScript

## 本地开发

直接在浏览器中打开`ss_app_prototype.html`即可预览。
